What are neurons and weights?

N

Neuron

A single computational unit. It receives inputs, multiplies each by a weight, sums them, adds a bias, then passes the result through an activation function.

W

Weight

A number on each connection that controls how strongly one neuron influences the next. Training adjusts weights to make predictions more accurate.

L

Layer

A group of neurons at the same depth. The input layer receives raw data, hidden layers learn representations, and the output layer produces the prediction.

What is Loss?

Loss measures the gap between the network's prediction and the correct answer. Common loss functions:

Mean Squared Error

L = (1/n) Σ (ŷ − y)²
Used for regression tasks.

Cross-Entropy

L = −Σ y·log(ŷ)
Used for classification tasks.

How the Network Learns

Backpropagation + gradient descent: the engine behind all deep learning.

The Learning Loop

1

Forward Pass

Feed input data through the network layer by layer to produce a prediction.

2

Compute Loss

Compare the prediction to the true label using a loss function. A bigger error = higher loss.

3

Backpropagation

Use the chain rule to calculate how much each weight contributed to the error — these are the gradients.

4

Gradient Descent

Nudge every weight in the direction that reduces the loss: w ← w − η · ∂L/∂w where η is the learning rate.

5

Repeat

Iterate over thousands of batches and epochs. Each pass sharpens the weights until the loss converges.

Image → Matrix → Neural Network

Every image is just a grid of numbers. Here's how pixels become the input vector a network sees.

Pixel Matrix (8×8)

Step 2 — Flatten into an input vector

The matrix is read row by row into a single vector of 64 numbers — the input layer of the network. Each number is then normalised to the range [0, 1] by dividing by 255.

A real image (e.g. 28×28 MNIST digit) produces 784 inputs; a 224×224 RGB photo gives 150,528. Convolutional layers learn spatial filters instead of flattening everything — but the principle is the same.

Arnold's Cat Map

A chaotic map that scrambles an image — yet applying it enough times always returns the original.

The Map

For an N×N image, each pixel at (x, y) moves to:

x' = (x + y) mod N
y' = (x + 2y) mod N

This is a linear map on the torus ℤN×ℤN. Because the pixel grid is finite, the orbit of every starting state is periodic — repeat the map enough times and every pixel returns home.
